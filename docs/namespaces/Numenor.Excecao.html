\Numenor

\NumenorExcecao

Namespaces

Php

Classes

ExcecaoAbstrata Classe abstrata de exceção utilizada pela biblioteca Numenor de forma que as exceções derivadas possam ser capturadas pelo comando catch() de forma diferenciada de outras exceções levantadas pelo sistema.
ExcecaoAlgoritmoHashInvalido Exceção levantada pelo sistema quando o nome do algoritmo de hash informado para a classe Checksum é inválido ou não está instalado na versão atual do PHP.
ExcecaoArquivoConfiguracaoInvalido Exceção levantada pelo sistema quando o arquivo de configuração da biblioteca informado não existe, não possui permissão de leitura ou não estpa em um dos formatos aceitos pelo leitor de arquivos de configuração.
ExcecaoAssetDuplicado Exceção levantada pelo sistema quando um asset já existente é adicionado à controladora de assets.
ExcecaoAssetNaoExiste Exceção levantada pelo sistema um asset é adicionado à controladora sem que exista o arquivo físico referenciado.
ExcecaoCacheDiscoDirEscrita Exceção levantada pelo sistema quando o adaptador de cache da classe CacheDisco é criado, mas o diretório de cache informado não possui permissão de escrita
ExcecaoCacheDiscoDirNaoDefinido Exceção levantada pelo sistema quando o adaptador de cache da classe CacheDisco é criado, mas o diretório de cache não foi definido (seja pelo bootstrap da aplicação, seja pelo parâmetro $diretorio no construtor do adaptador de cache)
ExcecaoChecksumChaveInvalida Exceção levantada pelo sistema quando a chave informada para a classe Checksum é nula ou inválida.
ExcecaoChecksumSemChave Exceção levantada pelo sistema quando a geração de um checksum é solicitada sem que a chave tenha sido definida anteriormente.
ExcecaoErroLogico Classe de exceções levantadas pelo sistema quando há um erro lógico na aplicação.
ExcecaoErroSistema Classe de exceções levantadas pelo sistema quando há um erro de sistema na aplicação.
ExcecaoErroUso Classe de exceções levantadas pelo sistema quando há um erro de uso de alguma funcionalidade da aplicação.
